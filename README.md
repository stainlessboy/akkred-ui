## Documentation

### General

* Make sure you have [node.js](https://nodejs.org/en/download/) and [yarn](https://yarnpkg.com/lang/en/docs/install/) installed
* $ yarn install
* $ yarn start
* go to [localhost:3000](http://localhost:3000/)


### Used Tools

* React.js
* React-Redux
* Universal-router
* Ant.design
* react-jss
